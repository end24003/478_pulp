# Option B Deployment Guide: Monte Carlo Simulation

## What You're Deploying

A web app that runs **1,000+ optimization scenarios with uncertainty** - demonstrating probabilistic thinking and risk analysis that Excel Solver cannot efficiently do.

**App Features:**
- Run 100-2,000 Monte Carlo simulations
- Account for uncertainty in demand, capacity, and prices
- Probability distributions and confidence intervals
- Risk analysis: best case, worst case, expected value
- Interactive probability calculator
- Downloadable simulation results
